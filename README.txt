написать программу согласно варианту для вычисления функции z = f(x).
Значения параметров a, b и аргумента x вводятся с клавиатуры.
Результаты вычислений выводятся на дисплей в формате с плавающей точкой.
Для ввода-вывода использовать scanf и printf.

Вариант 13.

z = |x| + sin(x), если x <= a
z = cos(|x|), если a < x < b
z = tan(x), если x >=b
